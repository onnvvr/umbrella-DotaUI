## 🖥 Troubleshooting

- **Loader.exe doesn’t start**  
Make sure your antivirus or Windows Defender does not block the file. Allow access if prompted.

- **Hotkeys don’t work in game**  
Check if umbrella runs with administrator rights. Right-click Loader.exe, select “Run as administrator,” and test again.

- **Overlays don’t show up**  
Verify that the game runs in windowed or borderless mode. Some fullscreen modes may block overlays.

- **Updates fail to download**  
Download new versions manually from the Releases page.

- **Performance drops**  
Close any heavy background programs. umbrella-DotaUI is designed to be lightweight but too many apps can affect performance.
